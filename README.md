mp3ファイルをWordファイルに書き起こすプログラム。

インタビュー調査などを行って録音を書き起こす際、安全性の面からウェブ上の書き起こしソフトを使えない人のために作成（臨床心理学の人など）。

GoogleのSpeech-to-Textというサービスを、speech_recognitionというAPIを通じて使用しています。このサービスのセキュリティについては以下を参照。

https://cloud.google.com/speech-to-text/docs/data-usage-faq?hl=ja 

自身のパソコンにPython環境があればpyファイルをダウンロードしても良いし、環境が無ければdistフォルダをダウンロードしてexeファイルを実行してください。
